# Design System

## Visual direction

Perplexity-inspired:
- clean
- narrow column
- information-dense
- low clutter

## Typography

- headings: Instrument Sans
- body: system stack

## Colours

- background: `#FAFAFA`
- surface: `#FFFFFF`
- text: `#1A1A1A`
- secondary text: `#666666`
- border: `#E5E5E5`
- accent: `#0066FF`
- success and savings: `#00A651`

## Layout

- mobile-first
- max content width `640px`
- 16px mobile side padding
- 12px radius cards

## Interaction rules

- 44px minimum tap targets
- skeletons before spinners
- transitions under 300ms

## Avoid

- decorative gradients
- heavy shadows
- rounded-full primary buttons
- emoji in UI
- stock photography
