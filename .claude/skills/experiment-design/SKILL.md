# Experiment Design

Invoke this skill explicitly when a task needs experimental treatment — do not auto-trigger from casual improvement requests.

Every experiment must state:
- baseline
- hypothesis
- single meaningful change
- metric
- keep threshold
- discard or reframe threshold

Log the result in `docs/reference/experiment-ledger.md`.
