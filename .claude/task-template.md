# Task Draft Template

Use this template before writing or rewriting a backlog item.

```md
- [ ] **[ID]** — [Action verb + specific behavior]
  - **File(s):** [exact file path or "new file: path"]
  - **What:** [one sentence describing the behavior change]
  - **Why:** [one sentence describing the user, ops, trust, or marketplace impact]
  - **Done when:** [specific outcome another person can verify]
```

## Pre-submit Checklist

- [ ] `Done when` is verifiable by someone who did not write the task
- [ ] no duplicate already exists in `todolist.md`
- [ ] file paths are exact, not broad globs
- [ ] this task describes one outcome, not a bundle of unrelated work
- [ ] the title uses a concrete action verb instead of `investigate`, `improve`, `enhance`, or `consider`
- [ ] the impact is explicit in `Why`
