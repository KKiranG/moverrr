# Agent Memory

This directory stores committed project memory for agents that use `memory: project`.

Guidelines:
- keep entries short and evidence-backed
- record recurring patterns, not one-off chat narration
- prefer file paths, invariants, and concrete failure modes over opinions
- do not delete memory casually; prune or rewrite stale entries when the repo truth changes

Project-scoped memory belongs here.
Personal or machine-specific memory belongs in `.claude/agent-memory-local/`, which is gitignored.
