# MoveMate Experiment Ledger

Use this file to record product, ops, or workflow experiments that should not disappear into chat history.

## Entry Template

### `EXP-YYYY-MM-DD-NN` — Short title
- **Baseline:** current number or current-state description
- **Hypothesis:** what should improve and why
- **Single change:** one meaningful variable
- **Metric:** what decides the result
- **Keep threshold:** what outcome means keep
- **Discard / reframe threshold:** what outcome means discard or revisit
- **Result:** what happened
- **Decision:** keep / discard / reframe
- **Notes:** uncertainty, follow-up, or what was learned
